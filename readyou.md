# Features

- Configure system settings using dedicated files like decoration.conf for font, cursor, blur, etc. 

* Hyprland
 - You can update the colors.conf file in ~/.config/hypr/colors.conf to change colors of the rice.

- kitty
 - Adds helpful keybinds
  | ctrl-backspace | delete word|
  | ctrl-shift-g   | copy output of last command|
 - Using Oh my zsh to manage plugins

- Neovim
 - Plugins and configs are managed in the nvim folder

# Package 

- oh-my-zsh
 - Run these two commands to install syntax highlighting and autosuggestions plugin
      git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
      git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
